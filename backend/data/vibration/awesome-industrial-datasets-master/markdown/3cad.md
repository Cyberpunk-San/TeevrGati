## Description

3CAD is designed for visual anomaly detection in 3C product inspection. It contains real industrial images across eight categories, covering normal samples and defective samples from practical quality-control scenarios.

The dataset provides 27,039 images with pixel-level annotations for anomalous regions. It can be used for image-level anomaly detection, anomaly localisation, and segmentation methods that require dense ground truth masks.

The dataset is distributed through the project GitHub repository together with paper and download information.
